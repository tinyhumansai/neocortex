# Realtime Personalized Call Assist

{% hint style="warning" %}
This use case is a **work in progress**. We're actively building and refining it with early partners.
{% endhint %}

Imagine a live call assistant that knows who you're talking to, what you've discussed before, and what matters to them by surfacing the right context in real time so you always sound prepared.

## How It Works

* **Personalized context on every call**: Before and during a call, the assistant recalls relevant history past conversations, preferences, action items so you never have to ask "remind me what we discussed last time."
* **Live suggestions**: As the conversation unfolds, the assistant surfaces talking points, objections the person has raised before, or commitments you've made all drawn from memory, not a static script.
* **Memory that sharpens over time**: Each call reinforces what matters. Topics that come up repeatedly gain importance; one-off small talk fades away. The assistant gets more useful the more you use it.
* **Cross-call continuity**: The assistant connects dots across calls if a prospect mentioned a budget concern three calls ago and a timeline last week, it brings both up when relevant.

## Example

A sales rep hops on a follow-up call. The assistant surfaces that this prospect cares most about integration speed (mentioned twice), has a Q3 budget deadline, and asked about SSO support last time. The rep walks in fully prepared without reviewing any notes.
