This folder contains all 3rd-party SDKs and integrations for the Neocortex memory API.

## SDKs

- [Python SDK](./sdk-python/README.md) - Neocortex Python SDK for building AI apps with persistent memory.
- [TypeScript SDK](./sdk-typescript/README.md) - TypeScript / JavaScript client for the Neocortex memory API.
- [Go SDK](./sdk-golang) - Go client for integrating Neocortex memory.
- [Rust SDK](./sdk-rust) - Rust client for integrating Neocortex memory.
- [Dart SDK](./sdk-dart) - Dart / Flutter client for integrating Neocortex memory.
- [C++ SDK](./sdk-cpp) - C++ client for integrating Neocortex memory.
- [C# SDK](./sdk-csharp) - .NET / C# client for integrating Neocortex memory.
- [Java SDK](./sdk-java) - Java client for integrating Neocortex memory.

## Plugins

- [LangGraph Plugin](./sdk-lang-graph/README.md) - LangGraph integration exposing Neocortex memory tools as LangGraph nodes / LLM tools.
- [OpenClaw Plugin](./plugin-openclaw) - OpenClaw plugin for using Neocortex memory inside OpenClaw workflows.
- [Vercel AI Plugin](./plugin-vercelai) - Vercel AI plugin for wiring Neocortex memory into AI routes.
- [LlamaIndex Plugin](./plugin-llamaindex) - LlamaIndex plugin for using Neocortex as a memory backend.
- [ElevenLabs Plugin](./plugin-elevenlabs) - ElevenLabs plugin for memory-aware voice experiences.
- [CrewAI Plugin](./plugin-crewai) - CrewAI plugin for agent teams backed by Neocortex memory.
- [Raycast Plugin](./plugin-raycast) - Raycast plugin for surfacing Neocortex memories from the desktop.
- [Agno Plugin](./plugin-agno) - Agno plugin for Neocortex-powered memories in Agno agents.
- [CamelAI Plugin](./plugin-camelai) - CamelAI plugin for Camel AI agents with long-term memory.
- [Pipecat Plugin](./plugin-pipecat) - Pipecat plugin for pipelines that persist and recall memory with Neocortex.
- [Mastra Plugin](./plugin-mastra) - Mastra plugin for Mastra workflows using Neocortex memory.
- [Keywords AI Plugin](./plugin-keywords-ai) - Keywords AI plugin for semantic memory and enrichment via Neocortex.
- [Dify Plugin](./plugin-dify) - Dify plugin for connecting Neocortex memory to Dify apps.
- [LiveKit Plugin](./plugin-livekit) - LiveKit plugin for real-time experiences with persistent Neocortex memory.
- [Autogen Plugin](./plugin-autogen) - Autogen plugin for multi-agent systems sharing Neocortex memory.
